# ISD Model for Online Learning

An interactive flowchart of an Instructional Systems Design (ISD) model. Click any milestone to view its purpose, subtasks, tools, deliverables, and AI prompts.

## How to publish this to the web (step-by-step)

### Prerequisites

- A GitHub account (free). Sign up at [github.com](https://github.com) if you don't have one.
- GitHub Desktop app (optional but recommended for beginners). Download at [desktop.github.com](https://desktop.github.com).

---

### Step 1: Create a new repository on GitHub

1. Go to [github.com/new](https://github.com/new)
2. **Repository name**: `isd-model` (or whatever you'd like)
3. **Description**: `Interactive ISD Model for Online Learning`
4. Set it to **Public** (required for free GitHub Pages hosting)
5. Check **"Add a README file"**
6. Click **Create repository**

### Step 2: Upload the files

**Option A: Using GitHub.com (easiest)**

1. Open your new repository on GitHub
2. Click the **"Add file"** dropdown button, then **"Upload files"**
3. Drag both files from the `isd-model-github` folder into the upload area:
   - `index.html`
   - `README.md`
4. Scroll down, type a commit message like `Add ISD model app`
5. Click **"Commit changes"**

**Option B: Using GitHub Desktop**

1. Open GitHub Desktop and clone your new repository
2. Copy both files (`index.html` and `README.md`) into the cloned folder on your computer
3. In GitHub Desktop, you'll see the new files listed as changes
4. Type a summary like `Add ISD model app` and click **"Commit to main"**
5. Click **"Push origin"** to upload

### Step 3: Enable GitHub Pages

1. In your repository on GitHub, click **Settings** (gear icon in the top menu)
2. In the left sidebar, click **Pages**
3. Under **"Build and deployment"**:
   - **Source**: select **"Deploy from a branch"**
   - **Branch**: select **"main"**, folder **"/ (root)"**
4. Click **Save**

### Step 4: Access your dashboard

1. Wait 1-2 minutes for GitHub to build your site
2. Go back to **Settings > Pages** and you'll see a green banner with your site URL:
   ```
   https://YOUR-USERNAME.github.io/isd-model/
   ```
3. Click the link. Your interactive ISD model is now live.

---

## Updating the model

To update the model in the future:

1. Make changes to `index.html` (or ask Claude to generate an updated version)
2. Upload the updated file to your GitHub repository (replacing the old one)
3. GitHub Pages will automatically rebuild within 1-2 minutes
