# ISD Model for Online Learning

An interactive, single-page dashboard that maps the full Instructional Systems Design (ISD) lifecycle for online learning projects. Built as a reference tool for instructional designers managing eLearning development from intake to project close.

## What It Does

The dashboard organizes the ISD process into 7 phases with 37 milestones, each containing sub-steps, tools and templates, deliverables, and AI prompts you can copy directly into Claude or ChatGPT.

**Phases:** Discovery, Analysis, Design, Development, Implementation, Evaluation, Project Close

**Features:**
- Click any milestone to view its details in a slide-out sidebar
- Tagged milestones: Meeting, Sign-Off, and Gate labels for quick scanning
- Linked tools and templates (OneDrive, Miro, Articulate 360, etc.)
- AI prompt library with one-click copy — includes meeting invite generators, pre-read document builders, storyboard creators, evaluation planners, and more
- Stat bar showing total phases, milestones, and quality gates

## Usage

Open `index.html` in any modern browser. No build step, dependencies, or server required.
